# TV Shows Explorer

Una aplicación moderna para explorar series de televisión consumiendo la API de TVMaze.

## Características
- Buscador de series en tiempo real.
- Visualización de detalles de la serie.
- Explorador de episodios por temporadas con código de colores según rating.
- Diseño minimalista, oscuro y responsive.
- Construido con Vanilla JavaScript (ES Modules).

## Tecnologías
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES Modules, Async/Await)
- TVMaze API

## Estructura de archivos
```
src/
├── services/   # Llamadas a la API
├── components/ # Componentes UI dinámicos
├── utils/      # Funciones de ayuda y constantes
├── styles/     # Estilos modulares
├── index.html
├── index.css
└── index.js
```
